# Обучающие карточки!

В этом стартовом проекте установлены следующие библиотеки

- **storybook**
- **fontsource**
- **radix-ui**
- **react**
- **react-dom**
- **vite**
- **typescript**

После git clone 'link' установите все зависимости из package.json:
### `pnpm install`

а потом запустите проект:
### `pnpm dev`
